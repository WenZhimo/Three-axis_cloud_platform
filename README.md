# STM32 三轴云台项目教材入口

本仓库包含 STM32F103RCTx 三轴云台项目源码、CubeMX/STM32CubeIDE 工程文件、Debug 构建产物，以及围绕该项目整理的中文工程教材。

阅读教材时建议优先进入：

- `教材正文/`：正式教材正文，共 34 章，适合按章节顺序系统学习。
- `教材设计/教材总纲.md`：查看全书篇章结构、读者定位和阶段目标。
- `教材设计/教材章节目录.md`：查看章节规划、前置知识和对应项目内容。
- `知识点/知识点总表.md`：查看正式知识点清单。
- `知识点/知识依赖图.md`：查看知识点之间的前置关系。
- `教材编写/术语与命名规范.md`：查看正文统一使用的术语、文件名和函数名规范。
- `教材编写/质量审查清单.md`：查看教材审查与出版优化标准。
- `教材编写/最终出版检查清单.md`：查看 Book Refinement 收尾前的最终审查标准。

## 教材正文阅读

正式章节位于 `教材正文/`，文件名按章节编号排序：

```text
第01章_MCU平台与项目边界.md
...
第34章_PID调参与系统优化路线.md
```

每章开头和结尾均包含上一章、下一章导航。每章正文按阅读层次组织：

- 【必须掌握】：后续章节依赖的主线内容。
- 【工程深化】：工程实现、调试经验和维护边界。
- 【拓展阅读】：帮助建立完整知识体系的补充内容。
- 【证据与验证】：项目源码、构建产物、官方资料或待验证项。

## 推荐路径

- STM32 初学者：优先阅读各章【必须掌握】内容，再按需要进入工程深化。
- 嵌入式开发者：重点阅读【必须掌握】和【工程深化】，并对照源码与构建产物。
- 算法读者：重点阅读姿态解算、滤波、PID、机械角/电角和系统控制章节。
- 维护人员：重点阅读调试方法、证据与验证、构建产物边界和硬件诊断章节。

## 证据边界

教材新增或强化的工程结论应来自仓库内证据，例如：

- 源码：`Core/`、`Drivers/`、`USB_DEVICE/`、`tools/`
- CubeMX/IDE 配置：`Three-axis_cloud_platformV2.ioc`、`.cproject`、`.mxproject`
- 构建产物：`Debug/Three-axis_cloud_platformV2.map`、`Debug/Three-axis_cloud_platformV2.list`、`.su`、`.cyclo`

凡涉及硬件实测、主机侧枚举、EEPROM 持久化、真实控制效果、温升、电机相序或长期稳定性的结论，如果仓库内证据不足，应继续保留 `【待验证】`。

## 仓库结构速览

```text

Drivers/        HAL/CMSIS、项目自定义驱动和控制算法源码

Debug/          当前保留的 Debug 构建产物与静态分析文件
tools/          辅助脚本，例如 PID 调参仿真工具
教材正文/       正式教材正文
教材设计/       总纲、章节目录和教学设计
教材编写/       模板、术语规范、质量审查清单和迭代记录
知识点/         知识点总表、依赖图、教学顺序和项目主线分析
```
