# 禁用或需谨慎使用的称呼

以下称呼不得替代正式名称：

- “板子”不得替代 STM32F103RCTx芯片平台，除非上下文明确指硬件整体且有证据。
- “定时器”不得替代通用定时器PWM输出、高级定时器TIM8、基本定时器TIM6。
- “姿态算法”不得替代四元数姿态解算或互补滤波。
- “电机驱动”不得替代三相正弦PWM电机驱动或三轴电机硬件映射。
- “串口”不得替代 UART串口调试输出。
- “USB通信”不得替代 USB FS设备、USB Device中间件或 USB CDC虚拟串口。
